정제된 병원 API 호출 문서

병원

요청 URL

/hospital/request/address/<major_address>/<minor_address>

요청 형식

# 항목명 항목구분 항목설명
1 address_major 필수 병원을 검색하고자 하는 위치의 도 이름입니다. ex) 충청북도, 경기도
2 address_minor 필수 병원을 검색하고자 하는 위치의 시군구 이름입니다. ex) 청주시, 성남시

응답 형식

병원 등급
  • A[상급종합병원, 종합병원]
  • B[병원, 군 병원]
  • C[의원,군 의원]
  • D[요양병원]
  • E[한방병원]
  • G[한의원,군 한의원]
  • H[약국]
  • I[기타]
  • M[치과병원]
  • N[치과의원]
  • R[보건소, 보건지소, 보건진료소, 보건의료원, S이송단체]
  • T[119구급대]
  • U[경찰서(교도소포함)]
  • V[지방자치단체]
  • W[기타시설, 군 기타]
  • Y[중앙응급의료센터]
  • Z[응급의료지원센터]
# 항목명 예시 항목설명
1 status true 달빛어린이병원 API 요청이 잘 전송되어 데이터를 얻을 수 있었는지 나타냅니다.
2 hospital_list list<dictionary> 달빛어린이병원 API 요청으로 받은 병원들의 정제된 데이터입니다.
3 hospital_list.address 2.66 해당 병원의 주소입니다.
4 hospital_list.level 의원 공공 API에서 규정한 병원의 등급입니다.
5 hospital_list.emergency_code_name 응급의료기관 이외 해당 병원의 응급의료기관코드명입니다.
6 hospital_list.emergency_room_available true 해당 병원에서 응급실을 운영하고 있는지 여부입니다.
7 hospital_list.information 운천내과와 주,야간 교대근무/주간:식도가시제거/수액○/사후피임약처방○/독감간이검사X 해당 병원에 대한 상세한 설명입니다.
8 hospital_list.information_etc 평일 18:00 ~ 23:00 토요일 일요일 및 공휴일 09:00 ~ 23:00 8월 14일 최신화 해당 병원 정보의 비고 항목입니다.
9 hospital_list.address_alias 진양아파트 맞은편 해당 병원의 추가적인 위치 정보입니다.
10 hospital_list.name 손범수달빛의원 해당 병원의 이름입니다.
11 hospital_list.contact 02-459-0119 해당 병원의 대표전화입니다.
12 hospital_list.latitude 36.65073962093224 해당 병원의 위도입니다.
13 hospital_list.longitude 127.47431470458069 해당 병원의 경도입니다.
14 hospital_list.start_time_mon 1800 해당 병원의 월요일 영업 시작시간입니다.
15 hospital_list.close_time_mon 2300 해당 병원의 월요일 영업 종료시간입니다.

18 hospital_list.start_time_wed 1800 해당 병원의 수요일 영업 시작시간입니다.
19 hospital_list.close_time_wed 2300 해당 병원의 수요일 영업 종료시간입니다.
