유치원 API 상세정보

URL : http://127.0.0.1:5000/children/{cityCode}/{pageNum}

요청 형식

# 항목명 항목구분 항목설명
1 cityCode 필수 도시 코드
2 pageNum 선택 페이지 번호

응답 형식

# 항목명 항목구분 항목설명
1 title 필수 유치원 이름
2 overview 필수 유치원 주소
3 telNo 필수 연락처
4 hpAddr 선택 홈페이지 주소
5 childNo 필수 총 유아 수
도시코드를 입력해주세요.
